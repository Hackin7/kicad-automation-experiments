› /plan provide me a plan on how to harness codex to effectively design a circuitboard. there are a few steps to it

  1. Schematic
    - Component selection and evaluation
    - Rough block diagram
    - Schematic netlist and connections
    - Schematic layout and organisation (laying out on the schematic sheet)
    - Simulation (if needed) and checking
  2. PCB
    - Mechanical dimensioning & board outline
    - Component Placement
    - Section component routing
    - Large Component routing
    - zones and planes
    - DRC checks
  3. Ordering
    - Generation

  There are some tools you can use to help you
  4. KiBot
  5. KiCAD-MCP
  6. KiCAD Python Library
  7. KiCanvas for ease of  viewing
